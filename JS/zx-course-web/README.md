# zx-course-web

> 口袋课堂WEB

## Build Setup

``` bash
# 1. 确认已正确启动 ZX-course-web-service 工程
Port: 8086
Context: /apicourse
开发环境下：登陆后需输入链接：http://localhost:8086/apicourse/login.jsp 再访问 http://localhost:8085

# 2. 确认已正确安装 node.js 和 npm
node -v （版本 > 6.9.3）
npm -v （版本 > 3.0.0）

# 3. 全局安装 vue 和 vue-cli
npm install -g vue
npm install -g vue-cli

# 4.下载安装依赖
npm install

# 5. 启动项目 （开发环境）
npm run dev / npm start

# 6. 构建打包测试环境
npm run buildtest

# 7. 构建打包生产环境
npm run build

# 8. 构建打包生产环境 （带有分析报告）
npm run build --report

# 9. run unit tests
npm run unit

# 10. run e2e tests
npm run e2e

# 11. run all tests
npm test
```
